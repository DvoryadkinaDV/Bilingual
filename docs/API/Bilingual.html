API для платформы Bilingual, которая объединяет пользователей для изучения иностранных языков

user

Методы управления пользователями

Получить список пользователей

Возвращает список пользователей, отфильтрованный по заданным параметрам

Authorizations:
None
query Parameters
language
string

Предпочтительный язык пользователя

skillLevel
string
Enum: "BEGINNER" "INTERMEDIATE" "ADVANCED"

Уровень владения языком

topic
string

Предпочитаемые тематики

Responses

https://api.bilingual.com/api/v1/users
